# 専門用語スキマ学習アプリ（仮）

忙しい社会人や新卒、未経験で新しい職種に入った人向けの、**専門用語学習アプリ**です。  
英語学習アプリというより、**「仕事で出てくる専門用語を、短時間で少しずつ覚える」**ことを目的としています。

---

## アプリの目的

- 本を読んだり長時間勉強したりするのは面倒だが、仕事で必要な専門用語は覚えたい人を助ける
- 通知やウィジェットを使って、スキマ時間に1問ずつ学習できるようにする
- 忘れやすい単語は繰り返し出し、少しずつ定着させる

## ターゲット

- 新卒
- 若手社会人
- 未経験職種に配属された人
- 忙しくて勉強時間を取りにくい人
- 英語や専門用語に苦手意識がある人

## コンセプト

- 専門用語を、選んだジャンルと言語ペアで学べるアプリ
- 学習は短時間、低負荷、毎日少しずつ
- 勉強感より「自然に覚える」体験を重視する

---

## 初期リリースの前提

- まずは **Android版** で開発
- 初期ジャンルは **IT・開発系**
- 初期ターゲットは **日本語話者の新卒・若手社会人**

## 必須機能（MVP）

- ジャンル選択
- 難易度選択
- 通知で専門用語を出題
- 通知上、または通知タップ後すぐの画面で3択クイズに回答
- ウィジェットで単語表示
- カタカナ読み表示
- 正解・不正解の記録
- 忘却曲線ベースの再出題（未定着語の重点復習）
- 出題された単語一覧の確認
- 正解単語／不正解単語の分類表示

## 出題イメージ

- 通知に英単語を表示
- 例: `workflow`
- 選択肢:
  1. 作業の流れ
  2. 職場
  3. 図表
- 回答結果に応じて、次回出題タイミングを調整

## 単語データで保持する情報

- 用語
- カタカナ読み
- 意味
- ジャンル
- 難易度
- 例文
- 例文の日本語訳
- 3択用の誤答候補
- 正解率
- 出題履歴

## デザイン方針

- 大人向け
- 幼すぎない
- 見やすい
- 朝や通勤中に片手で使いやすい
- 1回の学習は短く終わる
- 「勉強するぞ」という気合いがいらないUI

## このアプリの強み

- 通知から学習が始まる
- 専門用語に特化している
- カタカナ読み付きで、英語が苦手でも入りやすい
- 忙しい人向けに負荷が低い
- 苦手語を自動で重点復習できる

## このアプリで避けたいこと

- 一般的な英単語アプリ化
- 機能過多による重さ
- 初学者に難しすぎる説明
- 学習のために長時間アプリを開かせる設計

---

## 開発メモ（初期実装の方向性）

- 通知起点の1問学習を最優先で成立させる
- UXは「短い・迷わない・すぐ終わる」を基準にする
- 復習アルゴリズムは、初期はシンプルな忘却曲線モデルから開始し段階的に改善する
- データ構造は将来のジャンル追加・言語ペア追加に耐えられる形で設計する

## リポジトリ移行メモ（`SpecIterm` → `SpecItermS`）

`SpecIterm` の内容を `SpecItermS` に移すときは、履歴ごと移行できる **mirror push** を使う。

```bash
# 旧リポジトリを bare clone（履歴を含めて丸ごと取得）
git clone --bare https://github.com/taketakenon/SpecIterm.git
cd SpecIterm.git

# 新リポジトリへ履歴ごと反映
git push --mirror https://github.com/taketakenon/SpecItermS.git
```

移行後に必要であれば、旧リポジトリ側で以下を実施する。

- README に移行先（`SpecItermS`）への案内を追記
- 旧リポジトリを Archive 化（誤更新防止）
- GitHub Settings で default branch / Topics / Secrets を新リポジトリに揃える
